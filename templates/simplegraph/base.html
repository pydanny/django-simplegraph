Web Graph

What is this?

An easy-to-use tool for mapping out relationships between any types of objects you can image. The current working views are HTML with Graphviz generated images. Future views include CSV, 3-d animations, and more.

What is it built on?

A select group of entirely open source components supported by various groups such as Google, NASA, Microsoft, Sun, and the HITSS contract. Specifically, Python, Django, Graphviz, and Pydot.

How do I make changes?

You'll need to login and play around.
